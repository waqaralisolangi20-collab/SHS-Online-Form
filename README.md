# Sindh Solar Energy NGOs — GitHub Pages Template

یہ ایک professional front-end template ہے:
- 3-image automatic slider
- Urdu + English branding
- Responsive design
- NGO/community solar support registration form
- GitHub Pages compatible

## اہم حفاظتی نوٹ
یہ template جان بوجھ کر:
- CNIC/ID document upload نہیں کرتا
- کسی payment/IBAN collection flow کو شامل نہیں کرتا
- Gmail پر حساس معلومات خودکار طور پر نہیں بھیجتا

اگر آپ واقعی کسی NGO کے مجاز نمائندے ہیں تو پہلے organization کی قانونی شناخت، privacy policy، verified domain/email، اور secure backend تیار کریں۔ Payment یا حساس شناختی معلومات جمع کرنے سے پہلے واضح consent اور official verification ضروری ہے۔

## GitHub Pages
1. `index.html`, `style.css`, `script.js` کو repository میں upload کریں۔
2. Settings → Pages → Deploy from branch منتخب کریں۔
3. Branch `main` اور folder `/root` منتخب کریں۔
4. Save کریں۔
